# P3 バウル＆アルテマブラスター通し練習

 **バウル・オブ・アゴニー** と **アルテマブラスター** を通しで練習するためのページです。

バウル処理中に、次のアルテマブラスター判断へ切り替える練習ができます。

---

## できること

- バウル・オブ・アゴニーからアルテマブラスターまでの通し練習
- バウル中の各場面での立ち位置判断
- バウル後半からのアルテマブラスター突進・サイコロ判断
- サイコロ番号のランダム / 手動選択
- 実践テンポでのスタート・一時停止・再開
- 上部の `＜` / `＞` ボタンによる場面移動

---

## 対応している方式

以下の3方式に対応しています。

- ヤーン（TLB3）式
- ヤーン（DN / 正攻法）式
- Poikos式

初回表示は **ヤーン（TLB3）式** です。  
一度選択した方式は、ブラウザのキャッシュが残っていれば次回以降も復元されます。

---

## サイコロ設定

アルテマブラスターのサイコロ番号は、以下の2種類から選べます。

- **ランダム**
  - 問題ごとに1〜8番がランダムで選ばれます。
- **自分で選ぶ**
  - 1〜8番のうち、練習したい番号を固定できます。

初回表示は **ランダム** です。  
一度選択したサイコロ設定は、ブラウザのキャッシュが残っていれば次回以降も復元されます。

---

## 練習の流れ

1. 自分のロールを選びます。
2. 方式を選びます。
   - ヤーン（TLB3）式
   - ヤーン（DN / 正攻法）式
   - Poikos式
3. サイコロ番号を選びます。
   - ランダム
   - 自分で選ぶ
4. 「スタート」を押すと、3カウント後に通し練習が始まります。
5. 各場面で、マップ上の丸を選択して回答します。
6. 最後にアルテマブラスターのサイコロ散開位置を選択します。

---

## 実践練習の内容

### バウル・オブ・アゴニー

バウル中は、各方式に応じた立ち位置を選択します。

主な確認内容：

- MT組 / ST組分かれ
- 1回目の火水処理
- STのサンダガ誘導
- インプロージョン安置
- 2回目の火水処理
- D3のアンブラスマッシュ誘導
- 真空波 / TLB3付近の処理
- 最終集合・風の2人受け処理

### アルテマブラスター

バウル後半から、ケフカに見立てたピンクの丸が8方向から突進します。

- 8方向のいずれかから突進開始
- 時計回り、または反時計回りにずれながら8回突進
- 6回目の突進でサイコロ表示
- 最後に8方向の中間位置から正解を選択

サイコロ番号と突進方向をもとに、自分の散開位置を判断します。

---

## 操作方法

### 通し練習

- **スタート**
  - 3カウント後に練習を開始します。
- **一時停止 / 再開**
  - 進行中の練習を止めたり再開したりできます。
- **別の問題**
  - 新しい問題を生成します。
- **同じ問題をもう一度**
  - 現在の問題を最初からやり直します。
- **一問前に戻る**
  - 直前の場面に戻ります。
- **正解を表示**
  - 現在の場面の正解を表示します。

### スライド移動

マップ右上の `＜` / `＞` ボタンで、場面を前後に移動できます。

動画確認やタイミング調整を行うとき、特定の場面だけ確認したい場合に使えます。

## 注意

このページは個人制作の練習用ページです。  
実際の処理方法、採用マクロ、固定・野良の攻略法と異なる場合があります。

特に以下は、攻略情報の更新やPTごとの方針によって変わる可能性があります。

- ボス誘導位置
- サンダガ誘導の細かい位置
- 風の正面受け / 背面受けの細かい向き
- インプロージョン時の向き
- DN式・Poikos式の細かい立ち位置
- アルテマブラスターの判断方法や散開位置

実際に使用する際は、所属PTの処理法を優先してください。
